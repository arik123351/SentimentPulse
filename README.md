
Real-time Twitter sentiment analysis on stock tickers using Kafka, Spark, Elasticsearch, Kibana, and Docker.

## How to Run

```bash
docker-compose up --build
```

Then open:
- Kibana at http://localhost:5601
- Elasticsearch at http://localhost:9200
```
